<h1 align="center">Shelf with movie</h1>
<h3 align="center">Сайт-коллекция фильмов с системой рекомендаций на основе коллаборативной фильтрации</h3>
<br>
<h2>Функциональность сайта:</h2>
<ul>
  <li>Регистрация пользователя - окно с полями login,email,password,confirm password</li>
  <li>Авторизация пользователя - login,password</li>
  <li>Аутентификация пользователя во время авторизации</li>
  <li>Главная страница сайта - поле search,область рекомендаций фильмов</li>
 <li>При нажатии поля search появляется возможность ввода текста. При нажатии сбоку кнопки поиска поля search высвечивается список фильмов, схожий с введенным текстом пользователя. </li>
      <ul>
        <li>При нажатии на фильм из подборки рекомендаций открывается виджет с -</li>
            <ol>
                <li>названием фильма</li>
                <li>описанием фильма</li>
                <li>постером большего размера</li>
                <li>актерским составом </li>
                <li>рейтингом</li>
                <li>кнопкой просмотрен - при нажатии открывается окно оценки фильма: возможность поставить оценку от 0 до 10 и кнопка с сохранением результата в "просмотрен" коллекции</li>
                <li>кнопкой посмотрю - при нажатии добавляется в "посмотрю" коллекцию</li>
            </ol>
      </ul>
  <li>Страница коллекции пользователя - переход с главной страницы с помощью панели управления в зоне footer </li>
      <ul>
        <li>Вкладка "посмотрю" содержит список еще непросмотренных фильмов с возможностью нажатия на каждый фильм и перемещение во вкладку "просмотрен"</li>
        <li>Вкладка "просмотрен" содержит список просмотренных фильмов с возможностью нажатия на каждый фильм и удаление из вкладки "просмотрен"</li>
        <li>Отображение аватара пользователя и никнейма-логина пользователя. При нажатии аватара осуществляется переход в настройки профиля пользователя.</li>
        <li>Настройки профиля пользователя</li>
          <ul>
            <li>Изменение аватара пользователя</li>
            <li>Изменение никнейма-логина пользователя</li>
            <li>Кнопка выйти из аккаунта</li>
          </ul>
      </ul>
</ul>
